# 슬랙 알림 형식

```
🏢 해밀시아 임대점검 — YYYY년 MM월 DD일 (요일)

━━━ 🔧 오류 확인 필요 ━━━

📕 임차인마스터 (N건)
  ❌ [건물_호수_이름] — [오류 내용]

📙 이사예정관리 (N건)
  ❌ [건물_호수_이름] — [오류 내용]

📗 공실검증 (N건)
  ❌ [건물_호수] — [오류 내용]

📘 아이리스공실 (N건)
  ❌ [건물_호수] — [오류 내용]

📒 신규입주자 (N건)
  ❌ [건물_호수_이름] — [오류 내용]

(0건인 DB는 ✅ 표시)

━━━ ⚠️ 미납 확인 필요 ━━━

📛 미납리스크 (N건)
  🔴 긴급대처 N건 | ⚠️ 상습연체 N건
  🔵 입금대기 N건 | 🟡🟠 문자단계 N건 | ⬜ 기간내퇴거 N건
  ⚡ 오늘 입금예정: [이름들]

━━━ 💰 정산 진행중 ━━━

📚 퇴거정산서 — 진행중[입금대기] N건

━━━ 📌 미해결 항목 (N건) ━━━

⏰ [3일 경과] FLAT2_203 공실검증 미연결 — 담당: 이대리
⏰ [1일 경과] FLAT2_301 상태미입력 — 담당: 윤실장
🔴 [7일 경과] A동_404 계약서 미연결 — 담당: 윤실장 ← 에스컬레이션

━━━━━━━━━━━━━━━━━━━
오류 N건 | 미납 N건 | ⚡오늘 입금예정 N건 | 📌미해결 N건
```
