# 미납리스크 — 4분기 수식 기반 판정 (v3.0)

🚨[연체]확인필요 뷰 사용 가능 (checkbox + formula 복합 필터이나 작동 확인됨).
☑️입금완료 (checkbox)로 입금 상태 직접 판정, 계약납부일👤 (date)로 연체 여부 직접 계산.

**판정에 필요한 필드**:
| 필드 | 타입 | 용도 |
|------|------|------|
| Name | title | 식별 |
| 대응단계👤 | select | 긴급대처/상습연체자/입금대기중 등 분류 |
| 누적미납액👤 | number | 금액 표시 |
| 미납금액👤 | number | 현재 미납 금액 |
| 연체횟수👤 | number | 연체 횟수 표시 |
| 연체입금예정일👤 | date | 오늘과 비교 → ⚡오늘 입금예정 |
| ☑️입금완료 | checkbox | 입금 완료 여부 (formula 대체) |
| 계약납부일👤 | date | 연체 발생 기준일 (formula 대체) |
| url | url | 소스페이지 링크 |

**판정 순서**:

```
1. ☑️입금완료=true → ✅입금완료 (점검 대상 아님)
2. 계약납부일👤 또는 연체입금예정일👤 빈값 → ⚠️ 날짜입력필요
3. 오늘 > 연체입금예정일👤 → 🚨 입금날짜지남 (+N일)
4. gap = 연체입금예정일 - 계약납부일:
   - >30일 → 🚨🚨🚨 위험
   - >14일 → 🚨🚨 심각
   - >0일 → 🚨 주의
   - else → ⏰대기
```
