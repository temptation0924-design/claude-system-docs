# Screenshot Check

최근 스크린샷을 찾아서 보여주는 스킬.

---

## 트리거 키워드
- "스크린샷 찍었어", "스샷 찍었어"
- "스크린샷 확인", "스샷 확인해줘", "스샷 봐줘"
- "캡처 찍었어", "캡처 확인"
- "방금 찍은 스크린샷"

## 스크린샷 저장 위치
- `~/Documents/`

## 실행 절차

1. `~/Documents/` 폴더에서 최근 수정된 파일 목록 확인 (최신순)
2. 가장 최근 파일을 Read 도구로 열어서 보여주기 (이미지 파일은 자동 시각화)
3. 파일명, 촬영 시간, 파일 크기 정보 함께 안내
4. 필요 시 "이 스크린샷으로 뭘 할까요?" 질문 (분석, 저장, 공유 등)

## 복수 스크린샷
- "최근 스크린샷 3개" 같은 요청 시 → 최신 N개 나열 후 선택하게 하기
- 기본값은 가장 최근 1개
