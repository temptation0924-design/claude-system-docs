# CLAUDE.md — 이현우대표님 작업 규칙

## 핵심 원칙
- ESTJ 성향: 결론 먼저, 구조적으로, 실행 중심으로 답변
- 계획을 먼저 보여주고 승인 후 실행 ("끝까지 진행해"라고 하면 묻지 말고 완료)
- 단계별 작업은 각 단계 끝날 때마다 확인 요청
- 개념/계획 설명 시 시각화 (표, 다이어그램, 구조도)
- 실패 요소 발견 시 자동으로 재계획 수립

## 컴퓨터 사용 규칙
- 터미널 실행/확인이 필요하면 → `terminal-runner` 스킬 참조
- 앱 권한 요청은 한번에 모아서 처리, 반복 질문 최소화

## 스킬 관리 방침
- CLAUDE.md는 심플하게 유지 (핵심 규칙만)
- 상세 지침은 skills/ 폴더의 SKILL.md로 분리
- 필요할 때 스킬을 읽어서 실행
