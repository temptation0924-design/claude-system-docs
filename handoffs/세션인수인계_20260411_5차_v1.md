## 7. 주의 사항

### 7.1 SessionStart 훅 설계 교훈 (이전 실패 방지)
`ref-session-start-remind.sh`는 **session.md 내용을 절대 복제하면 안 됨**. 이전 훅이 2줄 요약을 주입했다가 Claude가 훅만 믿고 원본 무시 → B3/B4 반복 유발한 실패 사례의 근본 원인. 현재 구현은 "session.md 읽어라" 한 줄만 출력.

### 7.2 bash 3.2 호환성
macOS 기본 `/bin/bash` 3.2에서 동작해야 함.
- `declare -A` 미지원 → case 함수로 대체 (session-tracker-log.sh)
- `mapfile` 미지원 → `while read` 대체 (ref-dispatcher.sh)

### 7.3 `timeout` 명령 미지원
macOS 기본에는 없음. dispatcher가 `command -v timeout` 체크 + Python3 subprocess 폴백.

### 7.4 `--force-Bx` 탐지 소스
현재 구현은 `.user_message` → `.last_user_message` → `transcript_path` 폴백 체인. Claude Code 훅 입력 스키마가 실제로 어떤 필드를 제공하는지는 첫 실전 검증에서 확인 필요 (혹시 전혀 제공 안 하면 우회 기능 무력화 → 구현 조정).
