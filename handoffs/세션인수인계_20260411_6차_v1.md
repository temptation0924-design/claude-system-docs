## ⚠️ 주의/잠재 이슈

1. **노션 DB 위치**: 메인 대시보드 아닌 자료조사 페이지 아래. 기능적으로는 무관, 시각적/조직적 이슈만.
2. **Bash 3.2 호환성**: 이미 6번째 버그까지 잡았지만, 추후 새 코드 추가 시 `mapfile`, `declare -A`, `${var,,}` 같은 bash 4+ 문법 피해야 함. 또는 shebang 으로 brew bash 강제 (`#!/opt/homebrew/bin/bash`).
3. **단위 테스트 한계**: Notion 분기는 라이브 환경에서만 확인 가능 (NOTION_API_TOKEN 없는 unit test 환경에서는 분기 안 탐). 향후 mock 추가 검토할 만함.
4. **REF 와 독립**: REF 프레임워크와 충돌 없음. 둘 다 독립 운영 중.
