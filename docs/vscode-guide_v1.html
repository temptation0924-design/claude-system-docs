① 🔀 Git 브랜치 / ② ⚠ 오류·ⓘ 경고 / ③ Ln:Col 커서 위치 / ④ UTF-8 인코딩

💡 무엇인가요?

에디터를 닫지 않고도 "지금 어느 브랜치에서 작업 중인지", "오류가 몇 개 있는지"를 즉시 알려줍니다. 각 항목을 클릭하면 더 자세한 패널이 열립니다.

⌨️ 단축키

🖱 브랜치 클릭 → 브랜치 전환 메뉴
🖱 오류 숫자 클릭 → 문제 패널
🧪 실습 1분
  1. Status Bar에서 브랜치 이름을 찾아 클릭 → 전환 메뉴가 뜨는지 확인
  2. 오류·경고 숫자를 클릭 → 하단 문제 패널 열리는지 확인
  3. 커서를 이동하며 Ln/Col 숫자가 실시간으로 바뀌는지 관찰
✅ 성공 기준: Status Bar 4개 항목 이름과 역할을 설명할 수 있음

💡 무엇인가요?

Claude Code는 VSCode의 확장 프로그램입니다. 이미 설치되어 있다면 📦 확장 패널에서 이름과 버전을 확인할 수 있어요. 업데이트가 있으면 여기서 "Update" 버튼이 보입니다.

⌨️ 단축키

⌘⇧X → 확장 패널 열기
🧪 실습 1분
  1. ⌘+⇧+X로 확장 패널 열기
  2. 검색창에 "claude" 입력
  3. "Claude Code" 항목에서 현재 설치 버전 숫자 확인

💡 무엇인가요?

  3. 입력창에 "안녕" 입력하고 응답 받기
