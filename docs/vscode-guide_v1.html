~/.claude/settings.json은 Claude Code의 모든 자동화 규칙을 담습니다. 훅 등록, 허용된 명령 목록, MCP 서버 연결 등을 여기서 관리. 수정하면 Claude Code 재시작 시 반영됩니다.
