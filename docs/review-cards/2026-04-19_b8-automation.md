# 🎓 복습 카드 — B8 자동화 (debounce_sync.sh 근본 해결)

**📅 2026-04-19** | 🏷️ 자동 트리거 (시스템 설정 변경 + 새 패턴 + 에러 해결)

## 📌 한 줄 요약

68회 1위 누수였던 "Git 푸시 깜빡"을 **등기소 자동 팩스 시스템**으로 전환. 서류(커밋) 쌓이면 30초 후 자동 팩스, 보안 직인(시크릿 스캔)도 꼭 찍고 발송.

## 🎯 핵심 개념 (부동산 비유)

- **debounce 패턴** — 임차인이 10분 간격 5번 신고해도 15분 기다렸다 한 번에 출동. 타임스탬프 = 마지막 신고 시각.
- **mkdir atomic 락** — 관리사무소 열쇠 1개. 먼저 잡은 사람만 입장. flock 없이 POSIX 어디서나 작동.
- **시크릿 게이트** — 택배 엑스레이 검색대. 자동 push = 유출 고속도로, 반드시 스캔.
- **kill-switch (SKIP_B8_AUTOSYNC)** — 보일러 메인 차단기. 폭주 시 ENV 하나로 OFF.
- **macOS BSD ≠ GNU** — 서울 관리소장 규칙 ≠ 부산 규칙. `date +%N` 미지원 → python3 대체.
- **SESSION_ID stdin 전파** — 무전으로 "301호 작업중" 명시. `ls -t` 추정은 동명이인 위험.
- **Preflight = 2차 감리** — 매니저 혼자 놓친 OS 호환성·race condition을 3명 감리단이 46%→92.2%로 끌어올림.

## 🔄 과정

파일 저장 → `debounce_sync.sh` → mkdir 락 → 타임스탬프 기록 → 백그라운드 sleep 30s → 최신성 비교 → 시크릿 스캔 PASS → `git push` → 락 해제. Stop 훅 = 세션 종료 시 잔량 처리(fallback).

## ✨ 잘된 부분

- Preflight FAIL 무시 않고 CRITICAL 5건 수정 후 재검증
- kill-switch 선설계 (차단기 먼저)
- Stop 훅 fallback으로 엣지 커버
