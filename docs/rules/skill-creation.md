# 스킬 생성/관리 기록 규칙

## 감지 키워드
스킬, skill, SKILL.md

## 저장 위치
/mnt/skills/user/

## 동작
- 스킬 관련 대화가 감지되면 → 스킬 파일 생성/수정 절차를 따른다
- 새 스킬 생성 시 SKILL.md 표준 포맷을 준수한다

## 스킬 파일 구조
```
/mnt/skills/user/{skill-name}/
├── SKILL.md          # 스킬 정의 (필수)
└── (기타 지원 파일)
```

## SKILL.md 필수 항목
| 항목 | 설명 |
|------|------|
| name | 스킬 이름 |
| description | 한 줄 설명 (트리거 조건 포함) |
| 트리거 키워드 | 이 스킬이 활성화되는 키워드 목록 |
| 절차 | 단계별 실행 흐름 |
| 출력 형식 | 결과물 포맷 |

## 절차
1. 대표님이 스킬 생성/수정 요청
2. 기존 스킬 중복 확인
3. SKILL.md 표준 포맷으로 작성
4. /mnt/skills/user/ 에 저장
5. 필요 시 CLAUDE.md 기록 라우팅 규칙 테이블에 추가

## 주의사항
- 기존 스킬과 중복되지 않도록 먼저 확인
- 스킬 내에 민감정보 하드코딩 금지
- 대표님 확인 후 저장
