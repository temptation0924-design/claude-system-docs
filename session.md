## 슬랙 연동 원칙
- 슬랙 = 대표님 주요 채널, 모든 완료 알림은 슬랙으로
- 작업 완료 / Notion 저장 완료 / 에러 해결 시 슬랙 전송
- 형식: ✅ [작업명] 완료 / 📌 내용: (요약) / 🔗 Notion: (링크)

---

## 노션 기록 원칙
- 대화 분석 후 DB 자동 판단 → `📝 [DB명]에 저장할까요?` **1번만** 확인
- 오류/해결 → 에러로그 DB | 작업완료 → 작업기록 DB | 프로젝트 변경 → 프로젝트 현황 DB
- CLAUDE.md 수정 → 백업 페이지 | 프롬프트 발견 → 프롬프트 라이브러리 DB
- **임의 저장 절대 금지**

## 작업 완료 후 원칙

- 의문점 있으면 → 대답 + 더 나은 작업 방향 추천
- 세션 종료 전 → Notion 작업기록 DB 저장 (Claude팀장 MCP 직접)
